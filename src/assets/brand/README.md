## Placeholder design

The placeholder logos use a circle filled with the Libris gradient (`#3F1C66` → `#B12166` at 135°) with a white "L" centred in Inter 600.

## Replacing files

Drop the final files here with the same names. No code changes are needed — import paths in components reference these paths directly.

> Do **not** modify `build/icon.png`, `build/icon.ico`, or `build/icon.icns` here — those are managed separately in the Electron build pipeline.
